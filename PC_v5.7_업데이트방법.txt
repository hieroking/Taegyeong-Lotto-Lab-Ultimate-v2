太炅 Lotto Lab Ultimate PC v5.7

이번 통합 수정

1. 사진 인식 속도 개선
- 고해상도 사진은 OCR에 충분한 최대 1800픽셀로 자동 축소
- 같은 사진을 다시 인식하면 OCR 캐시 사용
- 사진 입력 후 중복으로 두 번 계산되던 현상 제거
- 여러 사진 처리 시 사진마다 추천 100조합을 다시 계산하지 않고 마지막에 한 번만 계산
- 특히 여러 장 사진을 넣을 때 체감 속도가 크게 개선됨

2. 신규 항목: 통합데이터추천
- 직접 입력한 번호
- 최근 100회 데이터
- 최근 500회 데이터
- 최근 1000회 데이터
를 실제 점수에 섞어 새로운 추천조합 100개 생성

3. 통합데이터 비율 5종
- 입력중심형: 입력 50% / 100회 20% / 500회 20% / 1000회 10%
- 최근중심형: 입력 25% / 100회 40% / 500회 25% / 1000회 10%
- 균형형: 입력 30% / 100회 25% / 500회 25% / 1000회 20%
- 장기형: 입력 20% / 100회 15% / 500회 25% / 1000회 40%
- 자동최적형: 최근 30회를 간이 검증해 위 네 방식 중 성적이 좋은 비율 자동 선택

4. 통합데이터추천에도 적용
- 필수번호
- 제외번호
- 후보번호
- 추천 이유
- 신뢰도·등급
- TOP10
- 상세 분석
- 유사조합 분산
- 역대 1등·2등 동일조합 제외
- Excel 저장

5. 기존 전략 배틀과 차이
- 전략 배틀: 전략들의 과거 성적을 비교하는 참고 기능
- 통합데이터추천: 입력번호와 100·500·1000회 데이터를 실제로 섞어 조합을 새로 생성

업데이트 방법
1. ZIP 압축을 풉니다.
2. GitHub 저장소의 Windows 폴더와 .github/workflows/windows-build.yml을 덮어씁니다.
3. Commit changes 후 Windows EXE 자동 빌드 결과를 받습니다.
