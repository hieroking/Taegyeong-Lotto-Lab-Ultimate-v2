太炅 Lotto Lab Ultimate v6.0

이번 버전에서 수정한 Android 빌드 오류
1. setContent import 누락 수정
2. Java와 Kotlin JVM 대상 버전을 모두 17로 통일
3. Apache POI가 요구하는 Android 최소 버전을 26으로 변경
4. compileSdk 35 경고 억제 설정 추가
5. Android APK 파일명을 v6.0으로 변경

GitHub 업로드 방법
1. 이 ZIP을 압축 풉니다.
2. 새 저장소의 Code 화면에서 Add file → Upload files를 누릅니다.
3. 압축을 푼 폴더 안의 전체 항목을 Ctrl+A로 선택합니다.
4. GitHub 업로드 화면으로 드래그합니다.
5. 아래 경로가 표시되는지 확인합니다.
   .github/workflows/android-build.yml
   .github/workflows/windows-build.yml
   Android/app/build.gradle.kts
   Android/app/src/main/java/com/taegyeong/lottolab/MainActivity.kt
6. Commit changes를 누릅니다.
7. Actions에서 Android APK 자동 빌드를 확인합니다.
8. 초록색 체크가 뜨면 Artifacts의
   TaegyeongLottoLab-Android-v6.0을 다운로드합니다.

기존 저장소에 덮어쓸 때
- 기존 파일을 삭제할 필요 없이 전체를 다시 업로드하면 같은 경로의 파일이 교체됩니다.
