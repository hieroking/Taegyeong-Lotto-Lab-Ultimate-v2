太炅 Lotto Lab Ultimate PC v5.5

v5.4 기준 추가·수정 내용

1. '같이 잘 나온 수' 명칭 수정
- '동반출현 횟수'로 변경
- 예: 3↔6 58회
- 58은 로또 번호가 아니라 두 번호가 역대 같은 회차에 나온 횟수입니다.

2. 추천 신뢰도와 등급
- 추천 신뢰도 0~100점
- S: 95점 이상
- A: 90~94.9점
- B: 80~89.9점
- C: 70~79.9점
- D: 70점 미만

3. TOP10 자동 강조
- 추천 상위 10개를 TOP 1~TOP 10으로 표시
- 표 전체 행을 별도 배경으로 강조

4. 추천조합 클릭 상세분석
- 동반출현 상위 5개
- 트리플 출현 상위 5개
- 홀짝·고저·합계
- 나온횟수·동반수·트리플·최근패턴·조합균형 점수
- 추천 이유와 신뢰도 등급

5. 조합 다양성
- 서로 5개 이상 번호가 겹치는 유사조합을 1차로 줄임
- 결과가 부족할 때만 점수순으로 보충
- 추천 100조합이 지나치게 비슷해지는 문제 완화

6. 기존 기능 유지
- 필수번호
- 제외번호
- 후보번호
- 추천 이유
- 10~14개 기본필터
- 15~24개 고급필터
- 25개 이상 정밀필터
- 자체추천
- Excel 저장

업데이트 방법
1. ZIP 압축을 풉니다.
2. GitHub 저장소의 Windows 폴더에 app.py와 requirements.txt를 덮어씁니다.
3. .github/workflows/windows-build.yml도 덮어씁니다.
4. Commit changes 후 Windows EXE 자동 빌드 결과를 받습니다.
