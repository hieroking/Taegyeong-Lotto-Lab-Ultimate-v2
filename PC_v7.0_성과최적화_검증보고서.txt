太炅 Lotto Lab Ultimate PC v7.0 — 성과 최적화 통합판

이번 버전의 중심
기능 수가 아니라 추천 결과 개선을 최우선으로 두고, 역대 1~1218회 데이터를 이용해 실제 30,000개 가중치 설정을 경쟁시켰습니다.

실제 수행한 검증
- 대상 데이터: 1~1218회
- 피처 생성 시작: 과거 300회가 쌓인 시점부터
- 테스트 설정: 30,000개
- 미래정보 차단: 각 목표 회차 직전 데이터까지만 사용
- 학습·검증·최종 보류구간 분리
- 마지막 보류검증: 1081~1218회
- TOP15 핵심번호 포함 성적을 중심으로 선택
- 무작위 TOP15 기대 포함번호 2.0개와 비교
- 특정 한 항목에 가중치가 과도하게 집중되는 설정 제한

최종 보류검증 결과
- 평균 당첨번호 포함: 2.123개
- 3개 이상 포함률: 31.9%
- 4개 이상 포함률: 14.5%
- 최고 포함번호: 5개

새 메뉴: 성과최적추천
1. 45개 번호의 성과최적 점수를 계산합니다.
2. 상위 후보번호 20개를 선정합니다.
3. 번호 선택과 조합 배치를 분리합니다.
4. 합계·홀짝·고저·구간·연속수·동반수로 조합을 재평가합니다.
5. 서로 5개 이상 겹치는 조합을 줄여 추천 100조합의 포착 범위를 넓힙니다.

데이터 관리
- 수동 회차 추가
- 중복회차·번호범위·보너스 중복 검사
- 저장 전 자동 백업
- 새 회차 추가 후 분석 캐시 갱신
- 최신 회차 공식 페이지 자동 확인
- 초기 안정화 기간에는 자동 결과를 화면에서 확인하고 수동 저장

성과 최적화 결과 확인
- 추천화면의 '30,000회 최적화 결과'
- 데이터 관리 > 성과 최적화 결과

중요한 해석
- 평균 포함번호는 핵심번호 TOP15 안에 실제 당첨번호 6개 중 몇 개가 들어왔는지를 뜻합니다.
- 추천점수와 합의도는 실제 당첨확률이 아닙니다.
- 과거 성적이 미래 당첨을 보장하지 않습니다.
- 로또는 무작위 추첨이므로 이 프로그램은 과거 데이터 안에서 상대적으로 우수한 설정을 찾는 분석 도구입니다.
