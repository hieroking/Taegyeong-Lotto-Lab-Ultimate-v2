太炅 Lotto Lab Ultimate PC v5.6

v5.5 기준 추가 기능

1. 추천조합 전략 6종
- 균형형: 조합 균형을 가장 중요하게 반영
- 출현형: 입력번호 출현횟수 중심
- 동반수형: 두 번호 동반출현 중심
- 트리플형: 세 번호 동시출현 중심
- 최근형: 최근 번호·최근 동반패턴 중심
- AI Ultimate: 출현·동반·트리플·최근·균형을 종합

2. 추천조합과 기존 항목 관계
- 전략 선택은 '추천조합' 항목에 적용
- 나온횟수·동반수·트리플·최근패턴은 기존 항목별 순위 유지
- 자체추천은 역대 전체 데이터 기준으로 독립 계산

3. 전략 배틀 백테스트
- 최근 100회를 대상으로 평가
- 각 회차의 이전 데이터만 사용
- 전략별 TOP10 중 최고 적중 개수를 비교
- 평균 최고 적중
- 3개 이상·4개 이상·5개 이상·6개 적중 횟수 표시
- 결과는 추천 화면 아래 상세창에 표시

4. 기존 기능 유지
- 필수번호·제외번호·후보번호
- 추천 이유
- 추천 신뢰도와 등급
- TOP10 강조
- 상세 분석
- 동반출현 횟수
- 조합 다양성
- 입력 개수별 자동 필터
- Excel 저장

주의
- 백테스트 결과는 과거 데이터 평가이며 미래 당첨을 보장하지 않습니다.
- 전략 배틀은 계산량이 많아 PC에 따라 1~3분 정도 걸릴 수 있습니다.

업데이트
1. ZIP 압축을 풉니다.
2. GitHub 저장소의 Windows 폴더에 app.py와 requirements.txt를 덮어씁니다.
3. .github/workflows/windows-build.yml도 덮어씁니다.
4. Commit changes 후 Windows EXE 자동 빌드 결과를 받습니다.
