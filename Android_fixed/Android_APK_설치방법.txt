太炅 Lotto Lab Android v1.0

휴대폰 기능
- 역대 당첨번호 Excel(.xlsx) 불러오기
- 사진 여러 장 선택 후 한국어 OCR 자동 인식
- 직접 번호 입력 및 OCR 결과 수정
- 추천조합 / 나온횟수 / 동반수 / 트리플 / 최근패턴 / 자체추천
- 각 항목 자동 100조합
- 합계 20~300
- 역대 1등·2등 동일 조합 제외

APK 만드는 방법
1. 이 ZIP을 압축 풉니다.
2. GitHub에서 새 저장소를 만들거나 기존 저장소의 별도 branch에 전체 파일을 업로드합니다.
3. Commit changes를 누릅니다.
4. Actions 탭에서 'Android APK 자동 빌드'가 완료될 때까지 기다립니다.
5. 최신 실행의 Artifacts에서 TaegyeongLottoLab-Android-v1.0을 다운로드합니다.
6. ZIP 안의 TaegyeongLottoLab-Android-v1.0.apk를 휴대폰으로 옮깁니다.
7. 휴대폰에서 APK를 눌러 설치합니다.
8. '알 수 없는 앱 설치 허용' 안내가 나오면 현재 파일 앱 또는 브라우저에 한해 허용합니다.

주의
- 현재 APK는 테스트용 debug APK입니다.
- 앱스토어 등록용 서명 APK는 별도로 만들어야 합니다.
- ML Kit OCR은 Android API 23 이상이 필요합니다.
- 사진 OCR 결과는 인식 오류가 있을 수 있으므로 번호 입력란에서 확인하세요.
