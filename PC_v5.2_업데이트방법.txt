太炅 Lotto Lab Ultimate PC v5.2

v5.1 기준 수정 내용

입력번호 개수별 자동 필터
- 10~14개: 기본 필터
  결과 확보를 우선하고, 결과가 부족하면 자동 완화하여 최대 100조합 생성
- 15~24개: 고급 필터
  v5.1의 홀짝·고저·연속번호 균형 기준 적용
- 25개 이상: 정밀 필터
  후보가 많으므로 합계와 연속번호 기준을 더 엄격하게 적용

공통 기준
- 번호합 최소 20, 최대 300
- 역대 1등·2등 동일 조합 제외
- 추천조합 / 나온횟수 / 동반수 / 트리플 / 최근패턴에 자동 적용
- 화면에 현재 적용된 필터 단계 표시

업데이트 방법
1. ZIP 압축을 풉니다.
2. GitHub 저장소의 Windows 폴더에 app.py와 requirements.txt를 덮어씁니다.
3. .github/workflows/windows-build.yml도 함께 덮어씁니다.
4. Commit changes 후 Windows EXE 자동 빌드를 확인합니다.
