太炅 Lotto Lab Ultimate PC v5.4

추가 기능

1. 필수번호
- 최대 5개
- 자체추천 제외 모든 항목에 적용
- 모든 추천 조합에 반드시 포함
- 일반 번호 입력란에도 포함되어 있어야 함
- 일반 입력 횟수에는 중복 합산되지 않음

2. 제외번호
- 최대 10개
- 자체추천 제외 모든 항목에 적용
- 추천 조합에서 완전히 제외
- 필수번호·후보번호와 중복 입력 불가

3. 후보번호
- 최대 10개
- 포함 시 번호당 4점 가산
- 최대 12점까지 반영
- 필수는 아니므로 포함되지 않아도 됨

4. 추천 이유 한 줄
예:
- 동반수 강함 / 최근패턴 우수
- 나온횟수 강함 / 조합 균형 우수 / 필수번호 3,6 / 후보번호 포함 7,14

5. 횟수 중복 방지
- 필수번호·제외번호·후보번호 입력란은 분석 횟수에 합산하지 않음
- 나온횟수는 일반 번호 입력란의 값만 사용

6. 기존 자동 필터 유지
- 10~14개 기본 필터
- 15~24개 고급 필터
- 25개 이상 정밀 필터

7. Excel 저장
- 필수번호
- 제외번호
- 후보번호
- 추천 이유
를 모두 저장
