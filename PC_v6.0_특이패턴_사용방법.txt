太炅 Lotto Lab Ultimate PC v6.0 — 특이패턴 엔진 통합판

핵심 신규 기능

1. 특이패턴추천
과거 패턴에서 이번 주 후보번호를 먼저 만들고, 후보번호로 추천조합을 생성합니다.

패턴 항목
- 이월수
- 2회 전 재등장
- 최근 10·30회 단기강세
- 장기 미출현 복귀
- 끝수 흐름
- 연속수 후보
- 직전번호 동반수 확장
- 최근 간격수 흐름
- 자동종합

2. 패턴투표
- 각 번호가 몇 개 패턴에서 추천됐는지 집계
- 번호별 패턴점수
- 번호별 추천 이유
- 패턴별 최근 과거 검증점수를 반영해 투표 가중치 적용

3. 이번 주 패턴 브리핑
- 패턴투표 핵심번호 TOP12
- 검증점수 상위 패턴
- 최근 10회 평균 합계·홀짝
- 제외 검토번호 8개
- 번호별 패턴투표 TOP20

4. 특이패턴 추천조합
- 패턴투표 상위 번호군으로 조합 생성
- 필수번호·제외번호·후보번호 적용
- 홀짝·고저·합계 균형
- 역대 1등·2등 동일 조합 제외
- 유사조합 자동 분산
- TOP10·신뢰도·등급·추천 이유 유지

5. 역대 유사 회차
추천조합을 클릭하면 아래 기준이 비슷한 과거 회차 TOP5를 표시합니다.
- 합계
- 홀짝
- 저·고 번호
- 끝수 종류
- 연속번호
- 번호 구간

6. 회차 검색
번호를 입력하면 해당 번호가 포함된 과거 회차와 일치 개수를 표시합니다.

7. 사진 인식 속도 개선 유지
- 고해상도 자동 축소
- OCR 캐시
- 여러 사진 처리 후 추천계산 1회
- 중복 재계산 방지

기존 기능 유지
- 추천조합 전략 6종
- 전략 배틀
- 통합데이터추천 100·500·1000회 혼합
- 필수·제외·후보번호
- 추천 이유·신뢰도·TOP10·상세분석
- Excel 저장

주의
특이패턴·신뢰도·검증점수는 과거 통계 분석 결과이며 실제 당첨을 보장하지 않습니다.

업데이트 순서
1. ZIP 압축을 풉니다.
2. 저장소의 Windows 폴더를 덮어씁니다.
3. .github/workflows/windows-build.yml을 덮어씁니다.
4. Commit changes
5. Actions의 Windows EXE 자동 빌드가 초록색으로 완료되면 Artifacts에서 받습니다.
